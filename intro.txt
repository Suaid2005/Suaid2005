# SET 1 — GIT

# Q1. Install Git and Configure Username & Email

## Aim

To install Git on the system and configure global username and email for version control.

---

## Step 1: Download Git

Open browser and visit:

[Git Official Website](https://git-scm.com?utm_source=chatgpt.com)

Download Git for Windows/Linux.

### Description

This step downloads the Git software.

---

## Step 2: Install Git

Run the downloaded installer.

Choose:

* Next → Next
* Use Nano editor
* Git from command line
* Install

### Description

This installs Git into the system.

---

## Step 3: Verify Git Installation

```bash
git --version
```

### Expected Output

```bash
git version 2.45.0
```

### Description

Shows installed Git version.

---

## Step 4: Configure Username

```bash
git config --global user.name "Muhammad Hamdan"
```

### Description

Sets global Git username.

---

## Step 5: Configure Email

```bash
git config --global user.email "hamdan@gmail.com"
```

### Description

Sets global Git email.

---

## Step 6: Verify Configuration

```bash
git config --list
```

### Expected Output

```bash
user.name=Muhammad Hamdan
user.email=hamdan@gmail.com
```

### Description

Displays saved Git configuration.

---

## Result

Git was successfully installed and configured.

# Q2. Create Multiple Directories using Loop

## Aim

To create multiple directories using loop.

---

## Step 1: Create Playbook

```bash
nano multi_dir.yml
```

---

## Step 2: Add YAML Code

```yaml
- name: Create directories
  hosts: local
  gather_facts: no

  tasks:
    - name: Create multiple dirs
      file:
        path: "/tmp/{{ item }}"
        state: directory

      with_items:
        - dir1
        - dir2
        - dir3
```

---

## Step 3: Run Playbook

```bash
ansible-playbook -i inventory.ini multi_dir.yml
```

---

## Step 4: Verify

```bash
ls /tmp
```

### Expected Output

```bash
dir1
dir2
dir3
```

---

## Result

Directories created successfully.

# Q2. Verify Title and Redirection

To verify page title and URL redirection.

---

```bash
nano verify_google.py
```

---

## Step 2: Write Code

```python
from selenium import webdriver

driver = webdriver.Chrome()

driver.get("https://www.google.com")

print("Title:", driver.title)
print("URL:", driver.current_url)

if "Google" in driver.title:
    print("Title verification passed")

if "google.co.in" in driver.current_url:
    print("Redirection passed")

input("Press Enter")

driver.quit()
```
